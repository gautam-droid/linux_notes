Moving Multiple Files with mv

1. Creating a Directory

Create a scripts directory: sh mkdir scripts

2. Moving JavaScript Files

Move all .js files into scripts directory: sh mv *.js scripts/

3. Verifying the Move

Check if the files are in scripts: sh ls scripts

4. Using -i for Safe Moves

Create a test file: sh echo "Test content" > test.txt

Check if home.html exists: sh ls home.html

If it doesnâ€™t exist, create it: sh echo "Home page" > home.html

Move test.txt with overwrite prompt: sh mv -i test.txt home.html

5. Additional mv Options

  • -f: Force move without prompt
  • -n: Do not overwrite existing files
  • -v: Verbose mode, explaining actions
